# war-robot
Terminal-based War card game written in C++17, designed with a hardware abstraction layer for a future robotic arm implementation.

## About
This project was built from the bottom up, starting with core mechanisms like shuffling and dealing before gradually moving to higher level abstractions like classes and game logic. Designed with embedded systems in mind, the codebase uses a hardware abstraction layer to keep the game logic hardware agnostic, with a future robotic arm implementation planned in a companion repository.

## Tech Stack
C++17, CMake 3.15+
